COVID-19 Data Insights
This project analyzes global COVID-19 trends using real-world data to uncover patterns in 
confirmed cases, deaths, and vaccinations. 
The analysis includes:
* Peak case, death, and vaccination counts per country.
* Mortality rate calculation based on the most recent data.
* Data cleaning and handling of missing values for accurate insights.
* Formatted, presentation-ready summary table for quick reference.
This project demonstrates:
* Data wrangling using Pandas
* Exploratory data analysis (EDA) techniques
* Real-world data handling with missing values
* Preparing professional Markdown outputs for reports & dashboards
The dataset was sourced from Our World in Data and analyzed using Python in Jupyter 
Notebook.
